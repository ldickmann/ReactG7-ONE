## 🔧 **Como Instalar e Executar o Projeto**  

1️⃣ Clone o repositório:  
```bash
git clone https://github.com/ldickmann/ReactG7-ONE.git
```  

2️⃣ Navegue até o diretório do projeto:  
```bash
cd ReactG7-ONE
```  

3️⃣ Instale as dependências:  
```bash
npm install
# ou
yarn install
```  

4️⃣ Execute o projeto:  
```bash
npm start
# ou
yarn start
```
